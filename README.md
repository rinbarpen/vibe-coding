# vibe-coding

## 技能库 (Skills)

本项目集成了以下技能库：

- **[Claude Scientific Skills](skills/claude-scientific-skills)**: 由 K-Dense AI 提供，包含 128+ 个科学研究领域的技能，覆盖生物信息学、化学、医学等。
- **[UI-UX Pro Max Skill](skills/ui-ux-pro-max-skill)**: 由 nextlevelbuilder 提供，专注于 UI/UX 设计智能化，支持多平台和技术栈的设计建议。
- **Anthropic Official Skills**: 从 [anthropics/skills](skills/anthropics) 集成，包括：
    - **Document Creation**: 支持 Word (`docx`)、PowerPoint (`pptx`)、Excel (`xlsx`)、PDF 处理及文档协同编辑 (`doc-coauthoring`)。
    - **[Frontend Design](skills/anthropics/skills/frontend-design)**: 用于生成具有高度设计感的前端 UI 界面和组件。
    - **[Theme Factory](skills/anthropics/skills/theme-factory)**: 为文档和应用生成及应用专业主题。
